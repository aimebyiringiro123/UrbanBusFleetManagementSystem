# Urban Bus Fleet Management System

A distributed Java application for managing urban bus fleet operations.

## Features

- Bus management
- Driver management
- Route management
- Ticket management
- Fleet tracking
- User authentication
- Database integration

## Technologies Used

- Java
- Java Swing
- MySQL
- JDBC
- NetBeans

## Project Structure

```bash
src/
 ├── controller/
 ├── model/
 ├── dao/
 ├── view/
```

## Installation

1. Clone the repository

```bash
git clone https://github.com/aimebyiringiro123/UrbanBusFleetManagementSystem.git
```

2. Open the project in NetBeans

3. Configure MySQL database

4. Run the project
